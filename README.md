Log medications, purposes, illnesses, dosages, and pill counts

Track remaining pills and receive refill alerts

Get daily notifications to take each dose

View the cheapest pharmacy prices for a medication by ZIP code

Enjoy an accessible, soft, healthcare-friendly UI with light & dark pastel themes
